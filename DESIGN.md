# ペット保険ナビ デザインガイドライン

## デザイン方針

**写真ベース**のリアルで信頼感のあるデザイン。イラストや絵文字は使用しない。

## カラーパレット

| トークン | 値 | 用途 |
|---|---|---|
| `--color-primary` | `#2B7A4B` | メイン緑・ボタン・見出し |
| `--color-primary-dark` | `#1f5c38` | ホバー時 |
| `--color-accent` | `#F5A623` | CTA・バッジ・強調 |
| `--color-accent-dark` | `#d4881a` | ホバー時 |
| `--color-text` | `#1a1a1a` | 本文 |
| `--color-text-light` | `#555` | サブテキスト |
| `--color-bg` | `#f9fafb` | ページ背景 |
| `--color-border` | `#e5e7eb` | 罫線・カード枠 |

## タイポグラフィ

- フォント: Noto Sans JP（Google Fonts）
- 本文: 16px / line-height 1.7
- 見出しH1: 2.2rem bold
- 見出しH2: 1.6rem bold
- 小テキスト: 0.85rem

## 写真・画像の使い方

### 基本方針
- **必ず実写写真を使用**する。イラスト・アイコン・絵文字は禁止。
- 写真はUnsplashなどのフリー素材を活用し、`public/images/` に配置する。
- `alt` テキストを必ず記述してアクセシビリティを確保する。

### 写真カテゴリ別ガイドライン

| 用途 | 方針 | ファイル例 |
|---|---|---|
| ヒーロー画像 | 犬または猫と飼い主の温かみある写真。明るく開放的な構図。 | `hero-dog.jpg` |
| ランキングカード | 犬種・猫種がわかるポートレート写真。白または淡いグラデーション背景。 | `dog-shiba.jpg` |
| 特徴セクション | 動物病院での診察シーン、飼い主がスマホを見るシーンなど。 | `vet-clinic.jpg` |
| 記事サムネイル | テーマに合った犬・猫の写真。16:9比率。 | `article-thumb.jpg` |

### サイズ・フォーマット
- ヒーロー画像: 1200×600px 以上
- カードサムネイル: 400×300px（4:3）
- フォーマット: WebP（フォールバック JPEG）
- 容量目安: ヒーロー≤150KB、カード≤80KB

### 写真素材の出典
- [Unsplash](https://unsplash.com/) — 商用利用可・クレジット任意
- [Pexels](https://www.pexels.com/) — 商用利用可・クレジット任意
- ダウンロード後、必ず WebP に変換・最適化してから `public/images/` に配置する。

## コンポーネント方針

### ヒーローセクション
- 全幅の写真を背景に、テキストをオーバーレイ表示。
- 写真には半透明の緑グラデーションをかけてテキスト可読性を確保。
- CTAボタンはアクセントオレンジ。

### ランキングカード
- 左端に保険会社ロゴ画像（ない場合は `<img>` プレースホルダー）。
- 絵文字・アイコンフォントは使用しない。
- 順位は大きな数字テキストで表現。

### 特徴カード
- 絵文字アイコンの代わりに写真または SVG アイコン（シンプルなライン系）を使用。

### FAQアコーディオン
- 装飾なし。Q/A の文字バッジのみ（現行デザイン維持）。

## 禁止事項
- 絵文字（🐾🐕🐈📊🏆💬 など）の使用
- イラスト素材・アバター・ステッカー風画像の使用
- stock photo 感が強い不自然なポーズの写真
- 無断使用・ライセンス不明の画像

## レスポンシブ対応
- ブレークポイント: 768px
- 写真は `object-fit: cover` で必ずアスペクト比を維持する。
- ヒーローはモバイルで画像を縮小し、テキストを前面に。
